## 02. 精准面诊：识别衰老的四大信号

在制定方案前，需通过面诊精准识别面部衰老的**“干、凹、松、垂”**四大核心特征，并定位其在皮肤解剖学中的层次。

| 衰老特征 | 视觉表现 | 皮肤层次/生理成因 |
| :--- | :--- | :--- |
| **干 (Dryness)** | 干纹、细纹、无光泽 | **表皮层/角质层**：屏障受损，水分流失，代谢减缓。 |
| **凹 (Hollows)** | 太阳穴/面颊凹陷、泪沟 | **皮下脂肪层/骨骼**：胶原蛋白流失，深层脂肪室萎缩，骨质吸收。 |
| **松 (Laxity)** | 弹性下降、捏皮回弹慢 | **真皮层**：弹性蛋白断裂，胶原纤维网崩塌，糖化反应。 |
| **垂 (Sagging)** | 法令纹、口角囊袋、下颌线模糊 | **筋膜层(SMAS)**：深层筋膜松弛，软组织受重力影响移位。 |

**【机体预警】**
衰老往往始于**炎症（Inflammaging）**。当皮肤底层出现超正常水平的泛红因子时，即便未见明显纹路，底层老化已在加速。

* **高阶套组：** 奢华黑金套 (¥1180) —— 全面抗衰护理。

### 第二层：医疗抗衰（院内操作 30%）
*目标：真皮层胶原再生，筋膜层复位，解决松弛与凹陷。*

 * **提拉复位：** 线雕或热玛吉/超声炮（针对下颌线模糊、苹果肌下垂）。

*目标：降低机体炎症，从源头延缓衰老。*

* **内服调理：** 补充胶原蛋白肽，内服抗氧化剂。
* **生活方式：** 严格控糖（抗糖化），减少熬夜（维持激素平衡），情绪管理（减少内耗）。
