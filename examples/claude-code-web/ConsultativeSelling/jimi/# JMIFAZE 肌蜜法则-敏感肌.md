# JMIFAZE 肌蜜法则·敏感肌专业面诊与调理手册

## 第一部分：敏感肌的基础认知与分级

### 1. 敏感肌普遍存在的4大类型
* **接触型敏感肌：**
 * **症状：** 紧绷、刺痛、发红、发痒。
* **环境刺激性敏感肌：**
 * **症状：** 紧绷发痒，容易出现红斑。
* **缺水性敏感肌：**
 * **症状：** 干燥脱屑，水油不平衡，细纹明显，易松弛老化。
* **发红性敏感肌：**
 * **症状：** 皮肤薄，毛细血管扩张。

### 2. 敏感程度的四度分级
* **轻度敏感：** 受刺激后泛红 + 敏感。
* **中度敏感：** 受刺激后泛红 + 紧绷 + 刺疼 + 起皮。
* **重度敏感：** 持续性泛红 + 紧绷 + 刺疼 + 灼热 + 红斑。
* **过激反应（激素依赖/严重受损）：** 持续性泛红 + 紧绷 + 发痒 + 灼热 + 红血丝（可能产生了皮炎）。

### 3. 敏感形成的诱因与过程
* **形成过程：** 皮肤受到刺激、化学物质、自由基、日光、寒冷、污染等影响，导致角质层受损，从「正常皮肤屏障」演变为「受损皮肤屏障」。
* **人为/后天诱发因素：**
 1. 过度清洁
 2. 不注意防晒
 3. 过度摩擦
 4. 季节温度变化
 5. 过度水合（敷面膜过频等）
 6. 刺激性化学成分
* **慎用成分提示（避雷区）：**
 * **皂基洁面：** 含十四酸、肉豆蔻酸、十二酸、月桂酸、十六酸等。
 * **防晒剂：** 二苯酮-3成分。
 * **防腐剂：** 尼泊金酯、甲基异噻唑啉酮、甲醛释放类防腐剂。
 * **其他：** 酒精、香精。

## 第二部分：敏感肌面诊卡（症状与成因诊断）

在面诊过程中，需根据客户面部特征进行对号入座：

### 1. 典型症状与表现
* **普通敏感肌：** 角质层薄有红血丝，遇热泛红发痒，皮肤干脱坡（遗传或屏障受损）。
* **大红脸（潮红）：** 长期面部潮红或间断性潮红（多因激素产品引起）。
* **红血丝：** 多发于颧骨或脸颊处，及鼻翼两侧成密集红血丝（遗传或屏障受损）。
* **脂溢性皮炎：** 皮肤发红起疹子，表面覆盖油皮屑或者结节发痒（内分泌、真菌、饮食导致）。
* **化妆品接触急性皮炎：** 突发症状，发红、瘙痒、灼热感、刺痛、丘疹、渗出（化妆品过敏或激素刺激）。

### 2. 深度成因分析
* **敏感肌：** 皮肤屏障功能受损、乱用产品、过度清洁。
* **大红脸：** 长期使用含激素护肤品及含激素药膏、超标强功效化妆品。
* **红血丝：** 内在原因（遗传、地域气候）；外在原因（皮肤屏障受损，反复炎症）。
* **脂溢性皮炎：** 饮食习惯、真菌感染、遗传、免疫下降、内分泌紊乱。
* **化妆品接触性皮炎：** 长期使用含激素护肤品/药膏、超标强功效化妆品。

 * **敏感类：** 激素脸、红血丝、血管网损伤。

 * **色斑/衰老类：** 黑黄粗、黄褐斑、真皮斑、干凹松垂。

## 第四部分：解决思路与调理方案

### 1. 总体解决思路
根据敏感程度不同，侧重点不同：
* **轻度敏感：** 补水为主，先建立健康的皮脂膜，同步重塑角质屏障。
* **中度敏感：** 先建立健康的皮脂膜，同步重塑角质屏障，最后抗炎。
* **重度敏感：** 先精简护肤，构建锁水屏障，再进行皮脂膜+屏障修护，最后抗炎。
* **过激反应（皮炎/激素脸）：** 先以温和舒缓为主，再做屏障修护，最后抗炎。

### 2. 标准管理流程
**清理 $\rightarrow$ 消炎 $\rightarrow$ 修护 $\rightarrow$ 补水保养**

### 3. 心理建设（重要！）
告知客户在调理受损肌底（特别是激素脸）时，可能会出现**“好转反应”**（排毒期）：如发红、肿胀、脱屑、渗出等。这是皮肤把底层炎症排出的过程，需耐心度过。

### 4. 调理方向与理念
* **消炎：** 理念是消除皮肤炎症。
* **修复：** 理念是修护皮肤屏障功能，重组皮脂膜。
* **保养：** 理念是选好修护、保湿、防晒保养品，减少熬夜及焦虑。

### 5. 方案设计阶段（分阶调理）
针对严重受损肌肤（如激素依赖性皮炎），通常分为三个阶段：
* **第一阶段：** 重点在于舒缓、撤药后的戒断反应应对。
* **第二阶段：** 重点在于深层消炎与屏障功能的初步重建。
* **第三阶段：** 重点在于养护，增厚角质层，恢复肌肤免疫力。

### 6. 居家搭配与注意事项
* **居家：** 必须配合诊疗方给出的居家修护产品。
* **重要提示（排毒期/好转反应）：**
 > **"皮肤底层本就受损，在调理的过程中，如出现皮炎唯有面对问题熬过去，才能焕然新生。"**
 * 若面部出现发红、肿胀、灼热、刺痛、干燥、脱屑、裂口或渗出液，通常是激素戒断或底层炎症排出的过程，需提前告知客户心理预期。
* **禁忌：** 避免过度清洁，禁止熬夜，慎用含刺激性产品。

大白话：
①长期盲目使用外用药膏，如皮炎平、肤轻松、艾洛松等激素类药膏涂抹于面部皮肤上导致皮肤免疫系统混乱 ，皮肤耐受力差变敏感。
②用过含有激素的祛斑、祛痘、美白等功效性护肤品，破坏皮肤屏障系统。
③由于皮肤一早对激素产生了依赖性，停药后原发病变反跳加重与继发的赤红、触痛、瘙痒、脱皮、干燥、裂口、脱屑等副作用。
这类皮肤的特点
1.面部皮肤发红、肿胀，伴随有明显的灼热感和刺痛感。
2.皮肤变得异常敏感，对许多日常护肤品都会产生过敏反应。
3.面部皮肤可能出现干燥、脱屑、紧绷等现象，严重时甚至会有裂口和渗出液。
4.激素脸患者的肤色往往不均匀，可能出现色素沉着或色素减退的现象，影响美观。

案例1：激素依赖性皮炎（泛红脱屑） 
背景：患者长期使用激素药膏，停用后爆发戒断反应。  
演练重点：
成交难点；我什么都用了就是不行？我不想试  
1.解释危害话术：如果是用药膏，激素像‘止疼药’，掩盖炎症但破坏屏障皮肤无法代谢吸收，如果是用冻干粉，只是表面吸收消炎，皮下吸收不了，所以需用古方疗，细胞更细，老坏更新掉，分层修复，逐步恢复。
2.推荐方案：3个月修敏方套组，重建皮质膜恢复皮肤代谢吸收。 
 案例2：反复过敏（红血丝+刺痛） 
背景：患者换季必过敏，化妆品不耐受。  
演练重点：  
1. 关联诱因：红血丝是屏障受损标志，需避免含酒精、香精成分的产品，只能用修护。材料产品如积雪草 唐蛋白 角鲨烷 油养肤  
2. 推荐方案：  3个月修敏方组合强化角质层重建
