## 一、 痘痘肌的基础认知与成因分析

### 1. 什么是痤疮（痘痘）？
痤疮即皮脂过多溢出堵塞毛孔而形成的一种皮肤问题。通常按照严重程度分为3度4级：
* **非炎性 轻度痤疮：** 闭口（I级）、粉刺（I级）
* **炎性 中度痤疮：** 丘疹（II级）、脓疱（III级）
* **炎性 重度痤疮：** 结节（IV级）、囊肿（IV级）

### 2. 痤疮形成的过程
堵塞毛囊形成粉刺 $\rightarrow$ 皮脂排除受阻，微生物繁殖诱发炎症（丘疹） $\rightarrow$ 炎症加重形成脓包 $\rightarrow$ 化脓或者更严重形成结节囊肿。

### 3. 痤疮形成的6大主要路径
1. **雄激素：** 刺激皮脂腺过度生长及分泌，导致毛囊角化。
2. **皮脂过度分泌：** 引发导管堵塞、菌群繁殖。
3. **痤疮丙酸杆菌增殖：** 脂肪酸含量迅速增长，诱发炎症反应。
4. **毛囊皮脂腺导管角化异常：** 堵塞导管，形成粉刺，进而演变成脓疱、结节。
5. **免疫细胞过度活跃：** 为细菌繁殖提供温床，导致痤疮持续扩展。
6. **缺乏维生素和锌：** 影响上皮细胞完整代谢。

**其他因素：**
* **外源性：** 高GI/GL饮食，使用了致痘致敏化妆品（如含油酸聚醚、棕榈酸、羊脂酸、月桂酸等成分）。
* **内源性：** 情绪压力、暴躁、易怒、上火、炎症体质。

## 二、 皮肤面诊与诊断流程

在制定方案前，需通过《痘痘肌诊断表》进行详细问诊：

### 1. 问诊关键点
* **过往史：** 是否用过激素类产品、中药膏、针清、刷酸等，停用时间多久。
* **身体状况：** 近期压力、熬夜频率、睡眠质量、生理期状况（痛经、血块）、消化系统（胃酸、便秘）。
* **饮食情况：** 是否偏爱冰饮、辛辣、甜食、油腻、烟酒。
* **护肤习惯：** 是否有防晒习惯、化妆习惯。

### 2. 皮肤类型评估
* **肤质：** 中性、混合型、敏感性、油性、干性。
* **皮损表现：** 闭口粉刺、白头/黑头粉刺、脓包痘痘、结节/囊肿痘痘、痘印、毛孔粗大等。

### 3. 常见成因总结
* **遗传因素。**
* **饮食不均衡：** 喝水少，蔬果少，肉/油/甜/辣摄入多。
* **内分泌：** 激素不平衡，皮脂腺分泌过多。
* **精神紧张：** 考试、升学、工作压力大。
* **其他：** 乱用化妆品、清洁不彻底、手挤痘痘、不注意卫生。

## 三、 解决思路与技术方案

### 1. 解决思路逻辑
* **非炎性痤疮（粉刺/闭口）：** 清理疏通 + 调节油脂 + 情绪饮食调节 + 修复屏障。
* **炎性痤疮（红肿/脓包）：** 清理疏通 + **抗炎抗菌** + 调节油脂 + 情绪饮食调节 + 修复屏障。

### 2. 现有祛痘方式优缺点对比
* **针清：** 见效快，但操作不当易感染留坑。
* **刷酸：** 疏通毛孔，但修复跟不上易脱皮敏感。
* **口服药：** 消炎快，但副作用大，需医生处方。
* **光电类：** 淡化痘印，但价格高，表皮损伤。
* **JMIFAZE方案优势：** 结合古方手法与科技产品，温和调理，修复屏障，避免反复。

### 3. 管理流程
**清洁 $\rightarrow$ 清理 $\rightarrow$ 消炎 $\rightarrow$ 修护痘印 $\rightarrow$ 控油保养**

### 4. 核心技术手法搭配
* **清甘油 + 无创六维净肤术 + 古方疗法**
* **诊疗方搭配：** 修敏方（焕颜美肌精华液）+ 祛痘方。

* **严禁事项：** 避免过度清洁、禁止熬夜、慎用含激素类产品。
* **饮食控制：** 必须控糖！避免牛奶、奶茶、辛辣和高糖类食物。
* **生活习惯：** 避免用手挤痘痘，保持良好的作息和心情。

**核心金句：** 痘痘肌做好后需保持良好的作息，及良好的饮食习惯、良好的心情，是不再长痘的真谛。
