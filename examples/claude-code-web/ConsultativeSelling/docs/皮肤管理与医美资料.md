## 2. 症状解码器 (用于 S2 诊断/嘴替)

> **指令：** 当用户输入左侧模糊词时，Agent 应调用右侧的场景描述进行“二选一”确认。

### 场景 A：关于"脸色/肤色"
| 用户模糊词 | 专家级场景拆解 (嘴替话术) | 对应诊断 |
| :--- | :--- | :--- |
| **"脸黄"** | **情况1 (油黄)：** T区爱出油，下午脸色暗沉，洗完脸能白回来，过一会儿又黄了。 | 油脂氧化 / 毛孔堵塞 |
| | **情况2 (暗黄)：** 像蒙了一层灰纱，洗脸也不变白，感觉是皮肤底色透出来的黄。 | 代谢迟缓 / 黑色素堆积 |
| **"没气色"** | **情况1 (枯)：** 皮肤干瘪，摸起来粗糙，上妆卡粉。 | 屏障受损 / 极度缺水 |
| | **情况2 (淤)：** 脸上有印子消不掉，唇色深，整体看起来累。 | 循环不畅 / 炎症色沉 |

### 场景 B：关于"老/垮/松"
| 用户模糊词 | 专家级场景拆解 (嘴替话术) | 对应诊断 |
| :--- | :--- | :--- |
| **"脸垮了"** | **情况1 (皮松)：** 捏起来软塌塌的，毛孔呈水滴状往下掉。 | 胶原流失 (表层) |

## 3. 解决方案矩阵 (用于 S3 方案设计)

> **指令：** 在 S3 阶段，优先输出结构化表格，遵循 "1.疏通 -> 2.代谢 -> 3.重塑" 的逻辑。

### 核心武器：7大诊疗方 (部分核心)

| 方案名称 | 核心动作 | 针对问题 | 用户利益 (FABE) |
| :--- | :--- | :--- | :--- |
| **① 清洁方** | **疏通** | 油黄、闭口、护肤品不吸收 | **"开门"**：把毛孔里的垃圾掏空，让皮肤能'呼吸'，解决洗不掉的'油黄'。 |
| **② 断黑方** | **代谢** | 底层暗黄、斑点、痘印 | **"扫地"**：击碎深层黑色素，加速代谢。让脸从'脏'变得'净'。 |
| **③ 减龄方** | **支撑** | 纹路深、凹陷、轮廓模糊 | **"装修"**：在骨骼流失的地方做支撑。不仅仅是填平皱纹，而是把脸'提'起来。 |
| **④ 修敏方** | **强韧** | 泛红、刺痛、红血丝 | **"加固"**：增厚皮肤屏障墙。让皮肤不再动不动就'发脾气'。 |
| **⑤ 水光方** | **灌溉** | 干纹、卡粉、外油内干 | **"浇水"**：把水打进真皮层。做一次相当于敷100张面膜。 |

### 经典组合策略 (Result-First)

**策略一：去黄提亮三部曲 (针对"脸黄")**
1. **清洁方**：先清掉表面氧化油脂 (去油黄)。
2. **断黑方**：再代谢底层色素 (去暗黄)。
3. **水光方**：最后补水提亮 (增加通透感)。

**策略二：抗衰紧致路径 (针对"脸垮")**
1. **清洁/调肤**：先把皮肤底子调理通畅 (确保后续项目吸收)。
2. **光电项目** (如热玛吉/超声炮)：收紧皮肉 (解决松弛)。
3. **减龄方** (针剂)：定点支撑 (解决凹陷/精修轮廓)。

## 4. 预期管理与异议阻击 (用于 S4 验证)

> **指令：** 遇到用户质疑时，调用以下数据和逻辑，展示坦诚。

### 关于效果的诚实分级
* **轻度问题 (100%)**：干纹、表层氧化黄、轻微毛孔粗大 —— **"一次见效，肉眼可见的改善。"**
* **中度问题 (70-80%)**：静态皱纹、真皮层斑点、轻度松弛 —— **"需要一个疗程（3-5次），能看到明显的年轻化，但不要指望变回18岁。"**
* **重度问题 (50-60%)**：严重凹陷、极度敏感、骨性衰老 —— **"我们可以改善现状，延缓衰老，但无法逆天改命，建议配合外科手术。"**

### 常见异议话术包 (Golden Sentences)

| 用户异议 | 核心逻辑 | 金句 (Golden Sentence) |
| :--- | :--- | :--- |
| **"太贵了"** | **价值锚定** | "贵不贵看效果。几千块的贵妇面霜涂一层如果不吸收，那是真贵；几百块的治疗如果能解决问题，那是真省钱。" |
| **"会有依赖性吗？"** | **原理科普** | "就像健身一样。你练好了身材，停下来肯定会慢慢回去，但这不叫依赖，这叫自然规律。皮肤护理是保养，不是换皮。" |
| **"会不会很痛？"** | **同理心** | "说一点不痛是骗人的。但我们的痛感管理做得很好，痛感如果是10分，我们能控制在2-3分，是可以接受的蚂蚁咬的感觉。" |
| **"你们会不会推销？"** | **品牌承诺** | "我们没有销售顾问，只有皮肤管理师。你的脸需要什么就做什么，我们不办卡，推销对我们没好处。" |
