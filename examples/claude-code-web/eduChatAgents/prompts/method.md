【唤学】教育体系

**1. 核心定位与理念**
- **本质：** 教育回归本质——**生命对生命的唤醒**。
- **核心策略：** **心学合一**（心理健康+学业成长深度融合）。
- **底层信念：** 无天生不爱学习的孩子，只有被阻碍的动力（缺乏被看见/理解/唤醒）。
- **核心洞察：** 学习的发动机是“心”（情绪/动力/价值观）而非“脑”。只有解决心理/动力问题，才能打破“学不进-挫败”的恶性循环。

**2. 教育模式对比（传统 vs 唤学）**
- **核心单元：** ❌学科（工业化流水线） vs ✅**学生**（个性化成长项目）。
- **终极目标：** ❌教会（应试/标准答案） vs ✅**让学生会学**（终身能力/解决问题）。
- **驱动力：** ❌外部驱动（分数/施压/恐惧） vs ✅**内部驱动**（好奇心/价值感/成就感）。
- **师生关系：** ❌权威与服从（单向） vs ✅**伙伴与导师**（双向/共创）。

**3. 唤学核心方法论**
**A. 学习流程重构（6步法）：**
`育心(激发内驱) → 动脑(主动提问) → 学(深度理解) → 示(输出内化) → 练(针对强化) → 考(能力验证)`
*差异点：先解决“心”的动力，再进行“脑”的学习；强调“教是最好的学”（示）。*

**B. 四阶成长体系（循序渐进）：**
1. **唤醒动力**：解决厌学/拖延，建立“我能行”的信念（从“要我学”到“我要学”）。
2. **结构化记忆**：解决碎片化/记不住，通过拆书/建模构建知识网络。
3. **系统化思考**：解决机械套用/点状思维，培养批判/举一反三能力。
4. **协同力**：解决眼高手低/发挥失常，实现眼-脑-手-心高效协同。

**C. 八步训练法（落地闭环）：**
`1.拆书(建框架) → 2.拆试卷(明目标) → 3.攻模块(破难点) → 4.慢读(深解) → 5.快读(强记) → 6.慢考(重过程) → 7.快考(提速) → 8.协同训练(稳输出)`

**4. 为什么要心学合一**
先育心，再动脑——心没打开，方法再多也白搭
为什么要封闭式集训营全托，因为心获得了能量，同步给予高效的科学学习方法论，并马上执行获得正向反馈，恢复自信心，心脑形成双向飞轮，加速成长。
相比传统的心理咨询，学生刚刚接受了心灵按摩获得了一些温暖，然而回到学校环境、回到家庭环境，行为照旧，环境给予的反馈没有变化，心灵温暖很快消耗殆尽。所以传统心理咨询往往都需要半年以上甚至一两年的持续服务过程，对于唤学的心学合一可以在一个月之内就取得显著改善的效果，传统的心理咨询师都觉得不可能。
